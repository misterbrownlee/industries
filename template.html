Howdy!

Throw a rock around here and you'll hit a web developer.

By here, I mean Silicon Valley. By web developer, I mean someone that can put you on the web. As a day job, I build high pressure, highly intricate internet applications. Sadly, this work has me narrowly focused on a single technology, and rarely involves HTML, JavaScript, or the other common tools used to create a simple, clean web presence.

tehfoo industries is my side work; it gives me a creative outlet, keeps me in touch with current trends in web development, and generates some scratch for my hobbies. If you ended up here, and if you need a web presence, we should talk. Since this is side work, I prefer to do simple projects, but it's likely I can help you in some fashion.
